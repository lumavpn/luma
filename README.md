# Luma VPN

This repository contains the source code for the Luma VPN app.

Please refer to our website luma.net for additional information how the service works.

## Features

- Proxy protocols: SOCKS5, HTTP(S), Wireguard
- Server selection
- Lightweight GUI
- Lightweight GUI
- Split tunneling
- Custom DNS
